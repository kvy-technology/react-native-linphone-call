# react-native-sip-phone

Make SIP calls from react-native using Linphone SDK

## Installation

```sh
npm install react-native-sip-phone
```

## Usage

```js
import { multiply } from "react-native-sip-phone";

// ...

const result = await multiply(3, 7);
```
